# .kiro

Workspace-level steering documents for the Nexus multi-agent learning lab, originally
authored for the Kiro IDE but treated as tool-agnostic architecture docs. The only
subdirectory is `steering/` — see `steering/AGENTS.md`. Nothing here is executed; these
are reference documents for humans and AI agents making cross-service changes.

If you are an agent working on a single service, that service's own AGENTS.md is more
specific; come here when a change spans services or you need the system-wide rules.
